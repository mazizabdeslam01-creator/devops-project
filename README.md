# DevOps Project — Network & Cloud Course

## Description
Application web Python Flask déployée avec une chaîne DevOps complète.

## Stack technique
- App : Python / Flask
- Conteneurisation : Docker
- Orchestration : Kubernetes (Minikube)
- CI/CD : GitHub Actions
- IaC : Ansible
- Monitoring : Prometheus + Grafana
